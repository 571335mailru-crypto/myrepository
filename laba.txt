ЛР1
> 1)Определите UID пользователя sys
> Ответ: UID = 3 (индентификатор)
> id -u sys
> или через grep
> cat etc/passwd | grep sys
>
>
> 2)Определите домашний каталог пользователя root
> Ответ: /root (6 поле)
>
>
> 3)Определите, какие пользователи принадлежат группе bin(2 способа)
> 1.Через функцию cat etc/group | grep bin
>
>
>
> 2.Через функцию getent group bin . Нет пользователя
>
>
> 4. Определите, принадлежит ли пользователь root группе other
> Проверяем на то, существует ли эта группа: getent group other
> ЕЕ не существует, если бы существовала, то ввели бы
>
> getent group other | grep root
>
> 5)Создайте нового пользователя c помощью утилиты useradd
> sudo useradd -m Alex5 (сразу с папкой создаст)
>
>
> 6)Установите личный недефолтный каталог пользователю user2 с помощью команды usermod
>
> 1.Создаем папку для нашего юзера, это будет ранее созданный Alex5
> с помощью команды sudo mkdir -p /opt/storage/Alex5
>
>
>
> 2.Меняем владельца, указываем Alex5
> с помощью: sudo chown Alex5:Alex5 /opt/storage/Alex5
>
>
> 3.Меняем запись в /etc/passwd с помощью usermod и опций -m -d
> sudo usermod -m -d /opt/storage/Alex5 Alex5
> 4.Проверяем
> grep Alex5 /etc/passwd
>
>
>
>
> 7)Задать пароль пользователю user2
> sudo passwd Alex5
>
